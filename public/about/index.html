About

PHYSLETS were developed at Davidson University by Wolfgang Christian. They are java applets that can be called from some javascript code in a web page. Unfortunately, these applets on http://vnatsci.ltu.edu/s_schneider/physlets/main/ only run on machines that have JAVA 8 or previous versions installed. In addition to this, most modern browsers do not support JAVA applets due to security vulnerabilities.
Therefore, we at PHYSLETS REVAMPED have recreated these simulations by utilizing modern HTML5 libraries, allowing them to be run on any device without any external software requirements.
Link to source code: https://github.com/rdeepak2002/PhysletsRevamped
